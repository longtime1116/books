# 舌のトレーニング
* 口を大きく縦に開ける
* 舌を突き出し引っ込める * 5
* 舌を突き出し、上下動かし * 5
* 舌を突き出し、左右動かし * 5
* 唇を閉じて舌を前に出し、左右動かし * 5

# 舌のホームポジション
* 舌を下につけた状態から少し手前に引き唾液腺に触るくらいな感じのところに置く(本当か？)
